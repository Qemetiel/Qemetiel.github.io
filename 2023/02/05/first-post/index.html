1 快捷键

(基于 Markdown All in One)

Command Key
加粗 CTRL + B
斜体 CTRL + I
删除线 ALT + S
$ 触发数学环境 $ CTRL + M
开启同步预览 CTRL + SHIFT + V
格式化表格 ALT + SHIFT + F

2 快捷命令

(基于 Markdown All in One)
CTRL + SHIFT + P 调出命令输入框，输入命令即可

Command Key
自动生成 Markdown 目录 Create Table of Contents
更新目录（没啥用） Update Table of Contents
添加/更新章节序号 Add/Update section numbers
移除章节序号 Remove section numbers
`添加code标签` Toggle code span
````添加code块```` Toggle code block
将当前文档打印到HTML Print current document to HTML
$ 触发数学环境 $ Toggle math environment
- 触发列表 Toggle list

3.1 字体

普通に(x 普通に(x

斜体 _斜体_
加粗 **加粗**
倾斜加粗 ***倾斜加粗***
删除线 ~~删除线~~
代码 `代码`

### 三级标题
#### 四级标题

===

3.6 引用

在被引用的文本前加上 > 以及一个空格，如：
> test
>
> test

引用可以嵌套：
> test3
>> test2
>>> test1

引用区块中 Markdown 语法仍生效。

3.7 列表

3.7.1 无序列表

“* “、”+ “、”- “ 均可得到无序列表，如
* test3
+ test2
- test1
效果

  • test3
  • test2
  • test1

3.7.2 有序列表

1. 使用数字
2. 和一个英文句点
3. 表示有序列表

效果

  1. 使用数字
  2. 和一个英文句点
  3. 表示有序列表

列表不光可以单独使用，也可以使用其他的 Markdown 语法，包括标题、引用、代码区块等。

3.7.3 检查列表

  - [ ] 计划任务
  - [x] 完成任务

效果

  • 计划任务
  • 完成任务
